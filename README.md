## [Oni_Recycling_Management](https://github.com/OriX0/Oni_Recycling_Management)

### 基本描述

由于公司业务需要，与奥尼厂家合作会免费寄送摄像头给到相关家长，如不报名，需进行回收，本系统是专门用于管理回收的系统。
